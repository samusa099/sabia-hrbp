# Sabia Group HRBP Analytics Package

A clean, structured package for **GitHub, Kaggle, Excel, Power BI, Python, SQL, and SQLite** practice.

## Start here

- `01_REFERENCE_DATA/` — company entities, departments, facilities, HR pillars, and data dictionary
- `02_RAW_DATA/` — deliberately messy files for data-cleaning practice
- `03_CLEAN_DATA/` — analysis-ready datasets grouped by business use
- `04_PROJECT_FILES/` — Excel workbooks, SQLite database, and Jupyter notebook
- `05_GUIDES/` — short usage and structure documentation
- `dataset-metadata.json` — Kaggle dataset metadata with column descriptions

## Recommended workflow

1. Review `01_REFERENCE_DATA/`.
2. Practice cleaning with `02_RAW_DATA/`.
3. Use `03_CLEAN_DATA/` for analysis and dashboards.
4. Open the relevant file in `04_PROJECT_FILES/`.
5. Use `dataset-metadata.json` when updating the Kaggle dataset.

> All records are synthetic and intended only for learning, practice, and portfolio demonstration.
