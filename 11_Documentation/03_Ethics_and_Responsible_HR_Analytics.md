# Ethical Use and Restructuring Guardrails

The dataset contains a difficult workforce-restructuring scenario for learning purposes.

Do:
- use job need, documented skill evidence, performance evidence and legal review;
- test adverse-impact risk;
- communicate criteria and appeal mechanisms;
- calculate separation benefits and knowledge-transfer risk;
- track workload and wellbeing of remaining employees.

Do not:
- use gender, age, religion, disability, ethnicity, pregnancy or other protected characteristics in selection;
- treat synthetic scores as sufficient for a real employment decision;
- infer that layoffs alone create profitability;
- present this project as a real company record.

All names and records are synthetic.
