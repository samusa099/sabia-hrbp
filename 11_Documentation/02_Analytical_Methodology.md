# Analytical Methodology

## Core analytical questions
1. Which workforce and organization problems contribute most to production loss?
2. What is the cost and risk of continuing, cost-only downsizing, pilot-then-scale, or immediate large-scale hiring?
3. Does the Q2 pilot materially outperform the control group?
4. Which skills, training and manager behaviours are associated with higher yield and productivity?
5. When does the business return to positive operating profit?
6. Which HR pillars reach enterprise-ready maturity by Q4?

## Suggested workflow
1. Load `05_Raw_Data` into Python or Power Query.
2. Apply the cleaning and validation script.
3. Compare outputs with `06_Clean_Data`.
4. Build a star schema using the model guide.
5. Reproduce the Excel executive dashboard in Power BI.
6. Add drill-through pages for workforce, talent, operations and financial impact.
7. Write an executive recommendation supported by Q1-Q4 evidence.

## Important modelling caution
This is a synthetic educational dataset. Associations do not automatically prove causation. The pilot/control comparison should be interpreted with operational context and sample-size limitations.
