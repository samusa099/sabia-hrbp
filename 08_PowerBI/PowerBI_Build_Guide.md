# Power BI Build Guide

1. Use **Folder** connector and select `06_Clean_Data`.
2. Create a Date table covering 2026-01-01 to 2026-12-31.
3. Apply the relationships in `model_relationships.csv`.
4. Hide technical keys from report view.
5. Use the DAX measures supplied in this folder.
6. Keep the executive page light: white background, navy title, teal success, orange risk.
7. Add a quarter slicer and drill-through from department to employee detail.
8. Add a tooltip showing the HR pillar, business metric and linked intervention.
9. Use a decomposition tree for operating profit drivers.
10. Add a test-versus-control page using `pilot_results.csv`.
