# Python Data Cleaning

## Environment

```bash
python -m pip install -r 07_Python/requirements.txt
```

## Cleaning workflow

```bash
python 07_Python/clean_and_validate.py
```

The script standardizes:

- employee names and gender values;
- department labels;
- dates;
- salary values;
- missing districts and skill levels;
- duplicate employee IDs;
- training status and costs;
- production rates and missing first-pass yield.
