2. Q1–Q4 headcount bridge তৈরি করুন।
3. Missing district, missing skill level এবং duplicate employee ID count করুন।
4. Production-এর monthly actual-versus-planned gap বের করুন।
5. Q4 operating profit বের করুন।

3. Performance score এবং training assessment-এর relationship।

## Advanced

5. Power BI-এর জন্য long-format KPI এবং HR-pillar views তৈরি করুন।
